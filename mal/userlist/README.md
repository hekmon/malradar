# MAL - User List

Undocumented public API

```
https://myanimelist.net/animelist/<USERNAME>/load.json?offset=<offset>&status=<status>
```

###	Offseting

Shows maximum 300 entries, showing the seconde page needs `offset=300`.

### Status filtering

* 1 == Watching
* 2 == Completed
* 3 == On Hold
* 4 == Dropped
* 5 == N/A
* 6 == Plan to Watch
* 7 == All
